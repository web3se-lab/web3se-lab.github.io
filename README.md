# SmartIntentNN Frontend

This is the official website and web tool demos for Web3 SE Lab.

**Website:** <https://web3se-lab.github.io>

**Backend:** <https://github.com/web3se-lab/web3-sekit>

## Install

```bash
yarn
# or
npm install
```

## Run Dev

```bash
yarn dev
```

## Build

```bash
yarn build
```
